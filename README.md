# 📖 LexiScope

## Semantic Text Similarity Analyzer

### Project Description
LexiScope is a Streamlit web application that measures semantic similarity between words, sentences, or short texts using the free pretrained NLP model **all-MiniLM-L6-v2** from Sentence Transformers.

The application does **not** perform preprocessing, text cleaning, tokenization, stemming, lemmatization, or model training. It directly uses the pretrained model to generate text embeddings and calculate similarity scores.

---

## Model Used

- **Model Name:** all-MiniLM-L6-v2
- **Library:** Sentence Transformers
- **Type:** Free Pretrained NLP Embedding Model

---

## Features

- Enter multiple words, sentences, or short texts
- Calculate semantic similarity
- Display similarity matrix
- Display top similarity pairs
- Bar Chart visualization
- Heatmap visualization
- PCA 2D Embedding Plot
- Paul's Critical Thinking Standards explanation

## How to Run

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
